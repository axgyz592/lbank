## Security: What's There, What's Not

LBank uses SSL encryption, cold wallet storage for the majority of assets, two-factor authentication (Google Authenticator or SMS), and risk control systems. The hot/cold wallet split is standard for reputable centralized exchanges.

What it doesn't have: Proof of Reserves publication, formal security audits from named third-party firms, or insurance coverage. These are legitimate gaps compared to top-tier exchanges like Kraken or Coinbase. They're worth knowing about. For most traders putting in a few thousand dollars to trade, the risk is manageable — but go in with eyes open.
